# AI Customer Support Ticket System

## Project Overview

AI Customer Support Ticket System is a Django-based helpdesk application that allows users to create support tickets and receive AI-generated responses using Google Gemini AI.

## Features

* User Registration
* User Login
* JWT Authentication
* Create Support Tickets
* AI Ticket Categorization
* AI Response Suggestions
* Ticket Status Management
* Admin Dashboard
* REST API
* Swagger Documentation
* PostgreSQL Database

## Technologies Used

* Python
* Django
* Django REST Framework
* PostgreSQL
* JWT Authentication
* Google Gemini AI
* Bootstrap 5
* HTML
* JavaScript

## Installation

1. Clone Repository

git clone YOUR_GITHUB_LINK

2. Create Virtual Environment

python -m venv env

3. Activate Virtual Environment

env\Scripts\activate

4. Install Requirements

pip install -r requirements.txt

5. Run Migrations

python manage.py migrate

6. Start Server

python manage.py runserver
